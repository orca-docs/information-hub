To provide a better user experience it is recommended to add to the *Orca Token List* appear in the UI.

A request can be made to add a token to the *Token List*, either when creating a new pool or at any time by talking to Orca via channels of communication your already have open, or by opening a ticket using the `Support` function in the wallet menu. You can also reach the community on [Discord](https://discord.gg/orca-so) or [Telegram](https://t.me/orca_so).
