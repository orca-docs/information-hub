# Orca's Public API

The Orca Public API provides programmatic access to Whirlpool data, token information, and protocol analytics.

## Quick Start

```bash
# Get protocol stats
curl "https://api.orca.so/v2/solana/protocol"

# Search for pools
curl "https://api.orca.so/v2/solana/pools/search?q=SOL-USDC"

# Get token info
curl "https://api.orca.so/v2/solana/tokens/search?q=ORCA"
```

## Base URL

| Chain | Base URL |
|-------|----------|
| Solana | `https://api.orca.so/v2/solana` |
| Eclipse | `https://api.orca.so/v2/eclipse` |
