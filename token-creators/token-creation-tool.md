With Orca's *Token Creation Tool* Orca expands on its vision to make DeFi more accessible and empower users with the ability to create tokens effortlessly and permissionlessly.

Whether youâ€™re a new project on Solana or Eclipse, community, this tool eliminates the complexities of token creation, making it as easy as a few clicks. This tool ensures that:

* **Accessibility**: No coding skills are required.
* **Customization**: Users can define token properties, including name, symbol, and supply

* **Security**: Mint authority and metadata mutability are protected.

### Step-by-Step Guide to Token Creation

* Navigate to https://www.orca.so/create-token or open the `Create` menu (yellow box) then click on `Create Token` (red box)

   * If you have not already done so, connect your wallet.
* Enter. descriptive name for your token

   * Choose a symbol/ticker for your token.

   * (Optional) Use the randomize button to change the *Token Mint Address*, you can use this repeatedly until you are satisfied.

   * (Optional) Provide a brief description for reference.

   * Drag and drop, or upload, an image to represent your token. Supported file types : SVG, JPG, JPEG, and PNG. The recommended format is a PNG with 256x256 pixel dimensions.

   * Choose a supply for your token, this cannot be changed later so ensure you have considered carefully. You can also click on the template supplies.

* When ready click `Preview Token`
* Review the details carefully and click `Create Token` once satisfied

    * You will need to authorize a transaction for each step of the token creation process




    * Congratulations, your token has been created!

    * (Optional) Click on Create Splash Pool to create your tokens first liquidity pool, you can find a guide [here](/token-creators/splash-pools) for that process.
